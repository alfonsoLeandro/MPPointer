Simple spigot plugin for allowing players to point to blocks for other players to see

### Usage
A player must define an item or items that will place pointers on the blocks they are looking at when right clicking with said item or items.

### Permissions
 - pointer.version - checks the current version
 - pointer.reload - Allows a player to reload the plugin
 - pointer.use - Allows a player to use pointers and set pointer items

### Commands
main command is  `/pointer ` or  `/pnt `
 - help - Shows a list of available commands
 - version - Shows the current version
 - reload - Reloads the plugin and its files
 - set - Sets the item held in the main hand as a pointer maker
 - remove - Removes the capacity for the item held in the main hand to make points
